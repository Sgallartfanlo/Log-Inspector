# Log Inspector

> Analitzador de fitxers de log per a Apache, Nginx i syslog.

Enganxa logs i obté estadístiques, errors, warnings, IPs recurrents i codis HTTP detectats automàticament.

## Tecnologies

- HTML + CSS + JavaScript (vanilla)
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Font Awesome](https://fontawesome.com/) (icons)
- [Fira Code](https://github.com/tonsky/FiraCode) + [Inter](https://rsms.me/inter/) fonts

## Funcionalitats

- Suporta logs Apache, Nginx i syslog
- Mostres de log predefinides per provar
- Comptatge d'errors, warnings i informació
- Detecta IPs recurrents amb recompte
- Detecta i classifica codis HTTP
- Visualització línia per línia amb codi de colors
- Fàcil de copiar i enganxar

## Ús

Obre `log-inspector.html` al navegador, enganxa el teu log al camp de text o carrega una mostra, i fes clic a "Analitzar".
